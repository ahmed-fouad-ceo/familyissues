# Family Conflict Solver — Arabic Prompt (AR)

برومبت عربي عملي لنزع التصعيد في الخلافات الأسرية:
- ✅ خطوات فورية (2–12 ساعة)
- ✅ شجرة قرارات (If X → A, else B)
- ✅ رسائل واتساب جاهزة
- ✅ خطة تحكيم 72 ساعة + خارطة 2–4 أسابيع
- ✅ يراعي الثقافة والدين والحدود الشرعية

## كيف أستخدمه؟
1) افتح `prompt_AR.md` وانسخه في أي دردشة ذكاء اصطناعي (حتى النسخ المجانية).
2) املأ الخانات بنقاط قصيرة وواضحة (بدون بيانات حساسة).
3) اطلب الرسائل الجاهزة والقوائم والملخّص صفحة واحدة.
